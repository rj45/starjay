# StarJay Fantasy Console Documentation

This is the documentation for the StarJay Fantasy Console SDK. This documentation is designed to help users understand how to use the SDK to create games or demos for the StarJay Fantasy Console.

- If you are a beginner and want a guided introduction: [Tutorials](./tutorials/README.md)
- If you need guidance on specific topics: [Guides](./guides/README.md)
- If you need detailed explanation of how the console works: [Manuals](./manuals/README.md)
- If you need quick reference: [Reference](./reference/README.md)


## Contributing

If you would like to contribute to the documentation, please feel free to submit a pull request. I welcome contributions of all kinds, including:

- Fixing typos, grammatical errors and inconsistencies
- Fixing factual errors, inaccuracies, outdated information or omissions
- Adding new content or examples
- Improving the organization and structure of the documentation

The documentation is structured according to [Diátaxis](https://diataxis.fr/), which is a framework for organizing documentation into four categories: Tutorials, Guides, Manuals, and Reference. Each category serves a different purpose and users with different immediate needs, making the documentation useful throughout the entire user journey, from beginner to expert.

### AI Use Policy

Please do not use AI to generate content for the documentation. You may use AI to advise you and help you orginize your thoughts, but the content itself should be written by a human. This is to ensure that the documentation is accurate, and because AI slop is easily recognized and turns people off. If you use AI, please explain how you used it, and why you believe it improved the quality of the documentation.

**All contributions that seem AI generated will be rejected.**

I may be wrong and mistake your human written content for AI generated content, and if so, I appologize in advance. If you believe your contribution was wrongly rejected, please reach out to me and I will review it again.
