# StarJay Zig SDK Template

You can copy this folder as the starting point of your StarJay project.

There are three build scripts included: `justfile`, `Makefile` and `Taskfile.yml`. Delete the ones you don't want to use.

There are also several `TODO`s in several files, you may want to search for them and fix them. They are mostly changing the name of the project from `starjay_template` to something else.

This template presumes you have done the equivalent of this:

```sh
git clone git@github.com:rj45/starjay-dev.git ../starjay
```

(For the public repo, remove the `-dev` from the URL.)
